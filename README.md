# 个人名片网页

## 作品名称
个人名片展示页

## 作品简介
一个简洁的个人介绍网页，展示基本信息、自我介绍、兴趣、技能等，可公开访问。

## 页面主要内容
- 姓名、学号、专业、联系方式
- 100字以上自我介绍
- 个性签名
- 兴趣标签、技能标签
- 个人照片
- 外部链接
- CSS样式排版

## 文件结构
- index.html    页面结构
- style.css     样式
- images/       图片
- README.md     说明

## 访问方式
发布到Gitee Pages后，通过公开链接访问。

## 体现软件作品的点
- 文件结构规范
- 样式与结构分离
- 响应式排版
- 可公开访问
